Introduction
============
This product adds an initiative content type and a viewlet to display the
five most recent initiatives on the front page.  Initiatives are basically just
News Items.

Initiatives must be published in order for them to show up on the front page

You can set a url of a collective to all the initiatives in the plone properties tool
Once set, a "more" link will show up below the initiatives on the front page

Other properties can be set in the site properties tool for initiatives
